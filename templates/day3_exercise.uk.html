  2. 11:30 — 12:30Фази 1–2: скафолд + моки. Кожна команда піднімає /health і mock-endpoint-и, щоб peer-и могли починати розробку проти вас.
  3. ~12:30Мок-чекпоінт. Кожна команда має fake-but-reachable версію себе. Немає мока = блокуєш увесь воркшоп.

ЗОЛОТІ ПРАВИЛА (прочитайте спочатку)
  1. Спочатку мок, потім реальне. Немає мока = блокуєте всіх downstream.
  2. Ваш інструмент теж є ціллю. Кожна команда робить адмін-сторінку з security-event трекінгом, окрему від основної ops-сторінки.
  3. Одна схема для всіх. Додаткові поля — ок; відсутні обовʼязкові — ні.
  4. Не передавайте секрети у промпти Claude. Використовуйте env-змінні, посилайтеся на імʼя змінної, не її значення.
  5. Health-endpoint у всіх. GET /health → 200 з назвою інструмента.
  6. Graceful-degrade на поганий вхід. Залогуйте, відмовте, поверніть 400, працюйте далі.
  7. Комітьте часто. Кожні 30 хв, навіть якщо поламано. Git — ваша кнопка undo.
